# Cinematic Image Prompt Template / 电影感图片提示词模板

Use this template when the user wants a cinematic, dramatic, or poster-like image.

当用户想要电影感、戏剧化、海报风格图片时使用此模板。

## Template
```text
{subject}, {core action or scene}, cinematic composition, dramatic lighting, rich atmosphere, detailed environment, expressive mood, high detail, sharp focus, professional color grading, 35mm film still, realistic textures
```

## 中文模板
```text
{主体}，{动作或场景}，电影感构图，戏剧化光影，氛围浓厚，环境细节丰富，情绪表达明确，高细节，清晰对焦，专业电影调色，35mm 电影剧照，真实质感
```

## Optional Add-ons / 可选增强
- Lighting: golden hour, neon backlight, soft window light, moonlight, volumetric light
- Camera: wide shot, close-up portrait, low angle, overhead shot, depth of field
- Mood: mysterious, heroic, calm, melancholic, tense, dreamy
- Style: realistic photography, anime film, concept art, editorial poster

## Example
Input:
```text
A samurai standing in the rain
```

Output:
```text
A lone samurai standing in heavy rain on an empty Kyoto street at night, cinematic composition, neon reflections on wet stone, dramatic backlight, misty atmosphere, detailed armor and fabric textures, solemn mood, sharp focus, professional color grading, 35mm film still, realistic textures
```
